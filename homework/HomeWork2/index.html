PHP та JavaScript

Мови програмування PHP та JavaScript дозволяє зробити сайт динамічним, тобто який реагує на дії користувача. Наприклад: можно зробити красиве випадаюче менб або слайдер

Види скриптів

Для цього пишуться скрипти (англ. script - «сценарій») - програми, як дозволяють реагувати на дії користувача. Скрипти бувають двох видів:

  • ті,які виконуються на сервері, а результат їх виконання приходить в браузер до користувача вже у готовому вигляді. Це скрипти написані на мовою програмування PHP. На ньому пишуться CMS-ки - системи управління контентом
  • ті, які виконуються прямо в браузері користувача. Це скрипти, написані мовою програмування JavaScript. Найчастіше вони використовуються щоб зробити сторінку більш зручною та красивішою
